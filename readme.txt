hyper-tracer 

Kinda like QIRA, but much much simpler.

Under 1.5K lines of code

Python analysis for deep execution and time-travel debugging

Current features:
- Real-time execution tracing
- Variable state tracking
- Memory & CPU profiling
- Function call graphs
- Control & data flow analysis
- Thread monitoring
- Bytecode analysis
- Web Interface for interactive visualization

Future:
- Pattern recognition in execution paths
- Custom rule creation for code analysis
- CI/CD pipeline integration
- Docker container support 
- Concurrency pattern detection
- Type inference
- 3D call graphs
- Heat maps for performance hotspots

Use cases:
- Debugging complex systems
- Optimizing performance
- Security analysis
